# Go-Ethereum Codebase 结构

为了更好的从整体工作流的角度来理解Ethereum，根据主要的业务功能，我们将go-ethereum划分成如下几个模块来分析。

- Geth Client模块
- Core数据结构模块
- State Management模块
    - StateDB 模块
    - Trie 模块
    - State Optimization (Pruning)
- Mining模块
- EVM 模块
- P2P 网络模块
    - 节点数据同步
- ...
